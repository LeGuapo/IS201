Why is coding so cool?

To know how to code opens many doors. You have the power of creation at your fingertips. The reason I love to code is because I view it as a puzzle to be solved.

There are so many possible solutions to a problem and it's up to the programmer to create the most elegant and simple one. A simple strand of code that enacts a complex process is a work of art.
